😸 TIP № 1: START FROM SCRATCH
Whether you are new to coding or not, the web-based tool Scratch is useful. You should know about Scratch. Scratch presents itself as visual blocks to clearly impart the logic of coding through playful actions akin to stacking legos. This makes learning code easy and great for beginners of all ages. For accomplished coders, Scratch presents itself as a useful tool for certain prototyping. That's how we do it. Human Angel Techprojects typically "start from Scratch."
Get started by playing and remixing these two remixable test games.
Go ahead. Start from scratch.

TIP № 2: COPY & CODE and REPEAT
After having "scratched the surface", next steps involve "removing the blocks" and seeing code as a language type, of which to manipulate and make meaning. Take your Scratch remixes to the next level by using Github to "fork" (copy) two more Human Angel Tech test web apps.
