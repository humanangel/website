Wanna Play a Game?

TIP № 1: START FROM SCRATCH

Whether you are new to coding or not, the web-based tool Scratch is useful. You should know about Scratch. Scratch presents itself as visual blocks to clearly impart the logic of coding through playful actions akin to stacking legos. This makes learning code easy and great for beginners of all ages.

For accomplished coders, Scratch presents itself as a useful tool for certain prototyping. That's how we do it.Human Angel Techprojects typically "start from Scratch."

Get started by playing and remixing these two Human Angel Techtest games made for you .Go ahead. Start from scratch.

TIP № 2: COPY & CODE and REPEAT

After having "scratched the surface", next steps involve "removing the blocks" and seeing code as a language type, of which to manipulate and make meaning. Take your Scratch remixes to the next level by using Github to "fork" (copy) two more Human Angel Tech test web apps.

Continue your learning by copying the code and making it all (reaonably) your own, on Github.

TIP № 3: BOOKMARK THIS ADVICE PAGE

litany of best practice resources or codepen io account...

Want to learn about the web? Try taking this digital literacy overview. Then, start with the Web Standards. Code with a free web editor like amaya and test your code with the World Wide Web Consortium validation tools. Passed the test? Get a badge. Go further by making sure your website is fair to people with visual and physical impairments by checking your file's accessibility or use the WAVE tool for each page. Keep your web project on your local server or launch it to a non-profit hosting service like Mayfirst.org. Learn to script your website with dynamic content using Ecmascript (Js) straight from the source or from Mozilla Developer's Network. Gain a better understanding of web graphics and the standards used to create, manipulate and optimize them. Brush up on your knowledge of HTTPS to better appreciate how the DOM, BOM and browsers work. Try some open source browsers like Firefox or Brave. Use a free search engine like Duck Duck Go and then try measuring the perfromance of your website with analytical tools like Matamo. Of course, you'd need to store your web assets in a convenient way and NextCloud can help, as well as Github. Reach out to us for a free consultation to learn and find what will feel right for you and your needs.
