⚠ The following 2 games are in progress and not intended to offend or harm.

  • GAME DESIGN: This game is more of a tool designed to help entreprenuers note their biases and allow them to clarify how their business should relate to people of varying age, ethnicity, gender, income, race, and sexual orientation. This project is intended to offend none, yet challenge each player to be as inclusive as possible. Questions are comprised of aggreggated elements that are multiplied by the number of each kind of person in the database (to be expanded). One of more than 1000 questions are randomly created by the game and presented to the player. Initially, a 60-second timer appears.

    HOW TO PLAY: Press 'play' and read the question presented. Then, select as many of the options that make sense as part of your answer before pressing play again to refresh page.

    View/Fork this project on GitHub.

  • GAME DESIGN: This project is designed to provide a bare astrology service, by separating a natal chart's interpolation from its interpretation. When prompted, the engine interpolates planetery positions for people born in the US between Jan 1 1950 and Dec 31 2020. This service stops short of creating a traditional astrological natal chart, instead providing only the planetary positions at the given time, without any meaning or inference.

    HOW TO PLAY: Fill out form with birth details (month, day, year, time of birth, AM/PM and place of birth) and submit.

    View/Fork this project on GitHub.

⚠ Learning 'code' can be fun, difficult, and sometimes both.
These 3 tips should help make the process easy, fun, and free.

TIP № 1:

START FROM SCRATCH 😸


Whether or not you are new to coding, the web-based tool Scratch is useful. You should know about Scratch. Scratch presents itself as visual blocks to clearly impart the logic of coding through animated drag and drop blocks. This makes learning code easy and great for beginners of all ages. For accomplished coders, Scratch presents itself as a useful tool for certain prototyping. That's how we do it. Human Angel Tech projects typically "start from Scratch." Get started by playing and remixing these two remixable test games. Go ahead. Start from scratch.

TIP № 2:

COPY & CODE ... and REPEAT 👨🏿‍💻


After having "scratched the surface", next steps involve "removing the blocks" and seeing code as a language type, of which to manipulate and make meaning. See how to create a countdown clock using CSS or how to create a random 'dice' like roll, by taking your Scratch remixes to the next level and using Github to fork (copy) games from Human Angel Tech.

TIP № 3:

GET ADVICE ... AND DEV TOOLS 💯

TIP № 4:

KEEP CRACKERS OUT!

Did you know that the most 5 common passwords are 123456  12345678  12345679  password and qwerty  even in 2022? That's weak, and weak passwords and weak password enforcement allows crackers (those who would maliccously attempt to crack your password the way a bank burglar would crack a safe) easy access to your data. But, that doesn't have to be so. There are ways you can protect yourself. Create a password at least 13 characters long. Include upper and lowercase letters and special characters. Change your password periodically. These are some of the actions that will bolster your password. You can also use information from the following sources to understand passwords and their protection.
